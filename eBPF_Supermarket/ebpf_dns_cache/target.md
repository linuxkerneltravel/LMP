# 评测

1. 实现项目预期功能
    - 缓存 DNS 解析记录及统计解析失败率
    - 在达到失败阈值时，例如有 20% 的 DNS Query 请求都失败了，可以尝试通过已缓存的解析记录，构造 DNS Reply 来解决解析失败的场景
    - 适用于容器场景 + IPV4
    - 与可视化项目对接（待可视化项目稳定后）
